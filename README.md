# Utiliser-les-commandes-Linux-pour-gerer-les-autorisations-de-fichiers
Projet réalisé lors de la certification Google Cybersecurity, mettant en pratique les notions en Linux..
# Contexte
Nous sommes professionnel de la sécurité au sein d'une grande organisation. Nous travaillons principalement avec l'équipe de recherche. Une partie de notre travail consiste à nous assurer que les utilisateurs de cette équipe disposent des autorisations appropriées. Cela permet de sécuriser le système.

Notre tâche consiste à examiner les autorisations de fichiers existantes sur le système de fichiers. Nous devrons déterminer si les permissions correspondent à l'autorisation qui devrait être donnée. Si ce n'est pas le cas, Nous devons modifier les permissions pour autoriser les utilisateurs appropriés et supprimer tout accès non autorisé.
